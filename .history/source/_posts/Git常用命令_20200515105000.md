### 常用Git命令
`git init`  初始化 在工作路径上创建主分支
`git clone` 地址 // 克隆远程仓库
`git clone -b` 分支名 地址 克隆分支的代码到本地
`git add -A` 保存所有的修改
`git add .` 保存新的添加和修改，但是不包括删除
`git add -u` 保存修改和删除，但是不包括新建文件。
所以默认使用git add -A就行
`git commit –m` "本次提交描述" commit可以一次提交缓冲区的所有文件，相当于一个版本。
<!--more-->

`git status`  查看状态
`git push` -- force
